# Introspecter

A framework for simulating and analyzing multi-agent LLM debates on reasoning benchmarks (GSM8k, MMLU, CommonsenseQA).

## Key Features

- **Automated Debates**: Orchestrate LLM discussions and observe consensus-building
- **Counterfactual Analysis**: Intervene in completed transcripts to test alternative outcomes
- **Benchmark Support**: Built-in integration with GSM8k, MMLU, and CommonsenseQA datasets

## Quick Start
