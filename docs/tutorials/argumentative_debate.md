# Argumentative Debates

Argumentative debates challenge agents to construct persuasive arguments grounded in evidence from a provided context. Unlike simple question-answering, agents must cite specific passages, build coherent reasoning chains, and respond to opposing arguments while maintaining logical consistency.

---

## Overview

This debate format guides agents through interpreting a context story, forming positions, selecting evidence, and constructing arguments. It is designed to evaluate reading comprehension, textual analysis, logical reasoning, evidence citation, and counterargument skill.

### Core Components

| Section | Purpose & Function |
|---------|-------------------|
| **Context Story** | Provides the narrative or source text that serves as the evidence base |
| **Debate Question** | Presents an interpretive prompt requiring argumentation and defensible positions |
| **Evidence Bank** | Contains numbered passages from the context for accurate citation and evidence selection |
| **Opposing Positions** | Establishes the contrasting stances participants defend or challenge |
| **Agent Positions** | Defines how AI and human debaters form arguments and cite evidence |

## Tips for Effective Argumentative Debates

- **Cite specifically**: Reference exact passages rather than general claims
- **Address counterarguments**: Don't just state your position; engage with opposing views
- **Build logical chains**: Connect evidence to claims with clear reasoning
- **Use context comprehensively**: Consider all relevant evidence, not just cherry-picked quotes
- **Maintain consistency**: Ensure your argument doesn't contradict itself across rounds
- **Track citations**: Use the evidence bank to see which passages have been used
